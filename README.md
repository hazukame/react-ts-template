# React TypeScript Template

Modern React starter template with TypeScript, Tailwind CSS, and Vite.

## Quick Setup

**Change site name & logo:**
1. Edit `index.html` - change `<title>` and favicon path
2. Add your `favicon.ico` to `public/` folder
3. (Optional) Edit `src/config/site.ts` for centralized config

See [SETUP.md](SETUP.md) for details.

## Stack

- React 18 + TypeScript
- Vite (Rolldown)
- Tailwind CSS
- React Router
- ESLint

## Usage

```bash
npm install
npm run dev
npm run build
```
