# React TypeScript Template

Modern React starter template with TypeScript, Tailwind CSS, and Vite.

## Setup

Change site title and favicon in `index.html`.

## Stack

- React 18 + TypeScript
- Vite (Rolldown)
- Tailwind CSS
- React Router
- ESLint

## Usage

```bash
npm install
npm run dev
npm run build
```
